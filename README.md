## Joe The Engineer

Welcome to Joe The Engineer game! :tada:

### Description

This project is a TOPDOWN shooter made on Blueprints with Unreal Engine 5.4

### How to Play

1. Clone the Repository:

```
git clone https://github.com/Rchi3-new/JoeTheEngineer.git
```

3. Open in Unreal Engine 5:
- Navigate to the cloned repository
- Open the project JoeTheEngineer.uproject

3. Build and Play:
- Once the project is open, build the project.
- Click on the *"Play"* button to start the game.

4. Controls:
- WSAD to move, F to interact, LMB to shoot, RMB to shoot with changed-shot.
